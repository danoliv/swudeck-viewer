## 🚀 Quick Start

1. **Start the server:**
   ```bash
   npm start
   ```

2. **Open the app:**
   ```
   http://localhost:8000
   ```

3. **Having CORS issues?**
   - Read: [CORS_FIX.md](./CORS_FIX.md) - Quick fix at the top!
